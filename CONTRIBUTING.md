# Contributing to WoT-Server

Thank you for your interest in contributing to WoT-Server. This document outlines how you can contribute and what to expect from the process.  

You can contribute in many ways, including:

- Reporting or fixing bugs
- Suggesting features or improvements
- Adding new features or integrations

You are encouraged to fork this repository and make any changes you like in your fork. 
Once your changes are ready, open a **Pull Request (PR)** to the `main` branch of this repository.

Please note:

- **Only the project maintainer can merge pull requests**
- Submitting a PR does **not** guarantee it will be merged
- PRs will be reviewed for:
    - Code quality
    - Maintainability
    - Security (especially important for IoT systems)
    - Alignment with the project’s goals

### Pull Request Guidelines

- Clearly describe **what** your change does and **why**
- Link any relevant issues
- Keep PRs focused and minimal
- Follow existing coding style and conventions

## Reporting Bugs

When reporting a bug, please include:

- A clear description of the issue
- Steps to reproduce
- Expected vs actual behavior
- Device, OS, firmware, or environment details
- Logs or screenshots if available

## Code of Conduct

Be respectful and constructive. Harassment, discrimination, or toxic behavior will not be tolerated.
